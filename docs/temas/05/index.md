# UT5 - **Desarrollo de Videojuegos 2D con Unity**

Bienvenidos a la unidad temática sobre el desarrollo de videojuegos. En este tema aprenderemos las bases para crear nuestros propios juegos en 2D utilizando **Unity**, uno de los motores más populares y potentes de la industria, y el lenguaje de programación **C#**.

Exploraremos desde el funcionamiento del motor y su interfaz hasta conceptos avanzados como el sistema de físicas, animaciones, prefabs y la gestión de la lógica mediante scripting.
